API Documentation

Getting Started

Our OCR API allows you to extract text from images and PDFs programmatically.

Authentication

All API requests require an API key. You can obtain one by registering on our portal.

Endpoints

POST /api/v1/ocr
Content-Type: multipart/form-data
Authorization: Bearer YOUR_API_KEY

Request Parameters

  • files - The file(s) to process (PDF, JPG, JPEG, PNG)
  • lang (optional) - The language of the text (default: auto)
  • output_format (optional) - Output format (json, txt, pdf)

Rate Limits

Free tier: 100 requests/day
Pro tier: 10,000 requests/day
